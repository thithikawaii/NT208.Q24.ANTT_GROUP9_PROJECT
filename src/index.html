CI/CD Flow

Pipeline mo ta

  • Push len `develop`

    GitHub Actions chay tu dong khi branch moi duoc day len hoac tao pull request.

  • Security scan va unit test

    Phat hien secret hardcode, loi code va regression truoc khi build image.

  • Build Docker va push registry

    Dong goi web app thanh image de su dung nhat quan giua local, CI va EC2 staging.

  • Staging deploy len EC2

    Pull image moi, chay container, sau do thuc hien smoke test hoac health check.

  • Rollback neu loi

    Khi endpoint bi loi 500 hoac health check fail, pipeline co the tra ve image tag truoc do.

Cac buoc demo

Checklist thuyet trinh

  • 1. Dang ky va dang nhap

    Tao user moi, luu xuong bang `users`, sau do dang nhap de vao khu task dashboard.

  • 2. CRUD task voi database that

    Them, sua, doi trang thai va xoa task de chung minh local va staging deu dung chung luong DB-backed.

  • 3. Mo phan debug insecure

    Xem thong tin DB host, DB user va password dang duoc app doc tu source/env de noi ve quan ly secret.

  • 4. Bam diem vao bi mat

    Kich hoat endpoint loi co chu dich de tra ve `HTTP 500`, lam dau vao cho rollback/smoke test fail.
